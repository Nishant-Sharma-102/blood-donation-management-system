# Bloodline

Bloodline is a **J2EE-based Blood Donation Management System** developed to manage blood donors, blood requests, and user authentication.  
The project demonstrates core **Java Enterprise Edition (J2EE)** concepts with a clean backend architecture.

---

## Technologies Used

- Java (J2EE)
- Servlets
- JSP
- JDBC
- MySQL
- Apache Tomcat
- HTML, CSS
- Maven (if used)

---

## Features

- User registration and login
- Donor management
- Blood request management
- Database connectivity using JDBC
- Server-side processing using Servlets
- MVC-based architecture

---

## How to Run

1. Clone the repository
2. Import the project into Eclipse / IntelliJ
3. Configure MySQL database connection
4. Deploy the project on Apache Tomcat server
5. Access the application via browser
